# 🌍 CountryBalls Life

**CountryBalls Life** သည် ကမ္ဘာ့နိုင်ငံများကို ပတ်ဝန်းကျင်နောက်ခံ သမိုင်းကြောင်းများနှင့်အတူ သရော်စာမြောက်မြောက် ကမ္ဘာလုံးပုံစံများ (Spheres) အဖြစ် ပုံဖော်ထားသည့် အဆုံးမဲ့ဇာတ်လမ်းကမ္ဘာ ဖြစ်သည်။ ပုံသေဇာတ်လိုက်ဟူ၍ မရှိဘဲ နိုင်ငံအသီးသီး၏ အလှည့်ကျလှုပ်ရှားမှုများကို စိတ်လှုပ်ရှားဖွယ် ရှုမြင်ရမည့် Ongoing Media Franchise ကြီး ဖြစ်သည်။
